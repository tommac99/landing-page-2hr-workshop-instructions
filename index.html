2 hour landing

Learn HTML and CSS with Le Wagon

1. Let's set up

  1. Start Visual Studio Code. It will open a New Window.
  2. Create a folder landing on your Desktop. Drag & drop this folder in the Sublime Text window.
  3. In VS Code Text left navigation:
    • Right click "New file"
    • Save it as index.html with Cmd + S or Ctrl + S
    • Do the same to create a styles.css file
    • Then "New Folder" and create an images folder
  4. Finally double click on index.html to open it with Chrome

2. Add some HTML content

  • To save time, just right-click on the following images and save them in your images folder
  • Then start with this HTML code in index.html
<html>
  <head>
    <meta charset="utf-8">
    <title>St Patrick's Day Event</title>
  </head>
  <body>
    <h1>St Patrick's Day Event</h1>
    <p>Feel Irish. Be lucky.</p>
    <a href="#">Buy tickets</a>

    <img src="images/shamrock.svg" alt="picture description" width="100">
    <h2>Shamrock</h2>
    <p>Feel the <strong>luck</strong></p>

    <img src="images/beer.svg" alt="picture description" width="100">
    <h2>Cold Beer</h2>
    <p>Taste the <strong>real</strong> beer</p>

    <img src="images/hat.svg" alt="picture description" width="100">
    <h2>Dress up</h2>
    <p>Get in <strong>Irish</strong> spirit</p>

    <img src="images/rainbow.svg" alt="picture description" width="100">
    <h2>Gold & Rainbows</h2>
    <p>Find your <strong>magic</strong></p>

    <p>Le Wagon - Coding Bootcamp - Amsterdam</p>
  </body>
</html>

Extra - find nice personal icons

  • You can pick your icons on IconStore, The Noun Project or NucleoApp (they have a free pack).
  • Tips: Also stay tuned checking icon resources on Product Hunt and icons8

3. CSS for fonts & colors

Link your stylesheet styles.css in the head section of your HTML

<head>
  <link href='styles.css' rel='stylesheet'>
</head>

Then copy/paste the CSS code below in styles.css and fix it cause it's ugly

  • For fonts use Google Fonts
  • For colors use Colorzilla Chrome plugin to pick colors from other websites
  • You can also find nice colors on Coolors or Color Hunt
/* style.css */
body{
  background: rgba(255,255,255,1);
  color: rgba(20,20,20,1);
  font-size: 24px;
  font-family: Helvetica;
  margin: 0;
}
h1{
  font-weight: 900;
  font-size: 40px;
}
h2 {
  font-size: 16px;
}
p {
  font-weight: lighter;
  font-size: 16px;
}
a {
  color: rgba(20,20,20,1);
  text-decoration: none;
}

5. Name your tags

  • Name your banner and footer with id like <div id="banner"> and <div id="footer">
  • Name your features with class like <div class="card">
  • Then add this nice CSS code
#banner{
  text-align: center;
  background-image: linear-gradient(
    0deg,
    rgba(0, 0, 0, 0.4),
    rgba(0, 0, 0, 0.6)
  ), url("images/banner.jpg");
  background-size: cover;
  padding: 80px;
  color: white;
  text-shadow: 1px 1px 5px black;
}
#banner h1 {
  font-size: 40px;
}
#banner p {
  font-size: 24px;
  font-weight: 400;
}
#footer{
  padding: 24px;
  color: rgb(49, 157, 17);
}
.card{
  background: rgb(219, 228, 207);
  box-shadow: 0px 0px 20px rgba(60, 90, 13, 1);
}

7. Bootstrap setup

Link Bootstrap stylesheet in the head section of your HTML before your stylesheet link

<head>
  
  ...

  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.0.0-beta2/dist/css/bootstrap.min.css" integrity="sha384-BmbxuPwQa2lc/FVzBcNJ7UAyJxM6wuqIj61tLrc4wSX0szH/Ev+nYRRuWlolflfl" crossorigin="anonymous">
  
  ...
  
</head>

Play with class="btn btn-light btn-lg" and class="list-inline" with class="list-inline-item", your first Bootstrap classes

We can over-ride the default Bootstrap card styling in our css

.card {
  background: transparent;
  border: none;
  font-weight: 400;
  text-align: center;
  margin-bottom: 20px;
}

.card img {
  width: 100px;
  margin: 0 auto;
  padding: 20px;
}
